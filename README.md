# Arduino 7-LED Chaser

An interactive 7-LED Chaser project built with Arduino Uno. 

## Project Overview
This project demonstrates sequential lighting control using Arduino. It serves as a foundational exercise in digital pin manipulation and circuit prototyping.

## Features
- Controls 7 LEDs in a sequential pattern.
- Uses optimized current-limiting resistors (270 Ohm) for safety and brightness.
- Simple, efficient code using arrays for easy expandability.

## Components
- 1x Arduino Uno
- 7x LEDs (Various colors)
- 7x 270 Ohm Resistors
- Breadboard & Jumper Wires

## Wiring
- Connect the LEDs to Arduino digital pins 2 through 8.
- Ensure the common ground (GND) is connected to the breadboard's ground rail.
